Interpolate Points

The Interpolate Points tool interpolates points into a raster surface.

Input point layer

The feature layer that defines the points.

Choose field to interpolate

Choose the field whose value to interpolate.

Output prediction errors

Choose the raster layer you wish to summarize.

Optimize for

Choose to optimize or not.

Clip output to

The layer to clip the output to.

Classify by

The classification type to use.

Number of classes

The number of classes to use.

Class break values

The values to break the classes into.

Predict at these locations

The locations to give predictions for.

Result layer name

This is the name of the layer that will be created in My Content and added to the map. The default name is based on the tool name and the input layer name. If the layer already exists, you will be asked to provide another name.

Using the Save result in drop-down box, you can specify the name of a folder in My Content where the result will be saved.
